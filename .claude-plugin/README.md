# b3nd plugin

Claude Code plugin carrying B3nd's vision and guidance. The skill teaches the
*shape* of the framework — three roles, message tuple, programs, schemas —
and instructs the agent to verify any concrete API or version against live
sources (JSR, GitHub) before generating code.

Install via the marketplace:

```
/plugin marketplace add bandeira-tech/b3nd-skill
/plugin install b3nd
```
